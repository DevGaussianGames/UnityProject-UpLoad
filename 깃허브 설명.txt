Git 이란?
형상관리시스템 (Verson Controll System)
주로 개발자들이 프로그램과 관련된 파일 저장
하는 데 사용 
게임 세이브 포인트와 유사한데 언제든지 저장
시점으로 돌아갈 수 있다.

저장소 클론(Clone )

세가지 기본개념 
커밋(commit):  

게임에 세이브에 해당 하는 행동을 git에서는 커밋이라고 한다.
다시 말해 언제든지  커밋한 시점으로 돌아 갈 수 있다.
커밋을 하려면 저장을 원하는 파일을 묶어서 커밋 명령을 수행 하면 된다.

스테이지 올린다(Add) 

앞에서 커밋 하기 전에 저장을 원하는 파일을 묶는 일 이작업을 스테이지에
파일을 올린다. 줄여서 간단히 Add라고 한다.

github 에 업로드 (Push)
커밋을 하면 이제 현재 작업내용의 세이브 데이터 내컴퓨터에 저장 되고
이걸 GitHub에 업로드 하면 마치 스팀이나 ps4 각종 모바일 게임의 원격 저장과
비슷한 일을 한다. 다른 사람이랑 공유 할 수도 있고 내 컴퓨터의 데이터가 날아가도
안전하게 다시 복구 할 수 있다. 

1. github 가입
2. github 저장소 생성
3. github 저장소 클론
4. 내컴퓨터 내문서 아래에 생성된 프로젝트 디렉토리에서 파일 생성및 작업 완료 
5. 커밋할 파일을 선택해서 스테이지에 올리기 :Add
6.Commit 하기
7. Push 하기
8. github.com/내아이디/내프로젝트 링크를 통해 정상적으로 업로드 되어 있는 지 확인
